-**Problema:** TSPTW.

-**Descripción:**
* `TSPTW` o mejor conocido como traveling salesman problem with time windows consta del problema de viajante pero con ventanas de tiempo. A partir de un recorrido determinado, hay que visitar todos los puntos una sola vez, de manera que debemos minimizar la distiancia, pero con la restricción de que solo pueden visitarse entre una hora mínima y máxima de llegada. Si llegamos antes, esperamos, si llegamos tarde, la ruta no es válida.
